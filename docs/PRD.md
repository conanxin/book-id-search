# PRD

## 目标

构建一个开源、可本地运行、可部署到腾讯云的图书元数据检索网站。真实 TXT 数据仅在本地或服务器私有导入，不进入 Git 仓库。

## 用户场景

- 通过书名、作者、出版社搜索图书。
- 通过 ISBN、SSID、DXID 精确定位图书。
- 查看单本图书详情与相关图书。
- 使用样例数据快速体验项目。
- 在服务器上导入真实 TXT 后提供 Web 检索。

## 功能范围

- TXT 流式导入 Meilisearch。
- 容错解析逗号和 Tab 分隔行。
- Express API 提供健康检查、搜索、详情、相关图书、统计接口。
- React 中文界面提供搜索、分页、详情和相关图书。
- Docker Compose 提供 Meilisearch、API、Web 服务。

## 非目标

- 不做登录注册。
- 不接入外部图书 API。
- 不把真实数据提交到 GitHub。
- 不做复杂权限系统。
