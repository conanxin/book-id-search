## 解析风险

- 真实全量数据仍可能存在少量缺 ISBN、缺页数、年份非数字、空作者、空出版社等弱解析行。

- 出版社字段中包含中文冒号，例如 `北京：中国纺织出版社`，冒号不是字段分隔符。
- 文件较大，导入时必须流式读取，不能一次性读入内存。

## 推荐解析策略

- 默认按 UTF-8、CRLF、逗号分隔解析。
- 若单行 Tab 数量达到 7 个且不少于逗号数，则按 Tab 分隔兼容。
- 使用容错字段映射：前两列固定为 `ssid`、`dxid`，后四列从尾部固定为 `publisher`、`year`、`pages`、`isbn`，中间字段尽量合并到标题和作者。
- `rawInfo` 永远保留，异常行不抛出进程级错误。
- `id` 优先使用 `ssid + "_" + dxid`，缺少任一字段时使用行号 fallback。
- 对缺字段、非数字年份/页数、缺 ISBN 等情况标记为 `weak`，仅严重缺失标题和关键 ID 时标记为 `failed`。
